# EduGenius AI - 智能教育助手

一个基于国产大模型的智能教育平台，为教师提供智能备课支持，为学生定制个性化学习路径。

## 功能特点

- 📝 教案生成：快速创建结构化、引人入胜的教案
- 📊 学习计划：为学生定制个性化的学习路径
- 💬 AI辅导：实时解答问题，提供学习指导
- 📈 成效分析：评估学习效果，优化教学策略

## 快速开始

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set the `VITE_DASHSCOPE_API_KEY` in `.env.local` to your DashScope API key

3. Start the development server:
   ```bash
   npm run dev
   ```
