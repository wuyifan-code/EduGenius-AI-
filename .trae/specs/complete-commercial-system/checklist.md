- [ ] 所有 UI 元素都可点击并有实际功能

- [ ] 下单流程完整（选择、确认、支付）

- [ ] 消息系统完整（发送、接收、实时）
- [ ] 通知系统完整（生成、查看、点击跳转）
- [ ] 支付系统完整（支付、退款）
- [ ] 管理后台完整（用户、订单、退款审核、统计）

### 用户体验
- [ ] 页面加载时间 < 3秒

- [ ] 错误提示友好且有帮助
- [ ] 移动端和桌面端都正常使用

### 稳定性
- [ ] 后端不可用时前端能降级使用
- [ ] 无 JavaScript 错误
- [ ] 所有 API 调用都有错误处理
- [ ] 数据库连接稳定

### 安全性
- [ ] 用户认证使用 JWT
- [ ] 敏感操作需要验证
- [ ] 支付流程安全
- [ ] 无敏感信息泄露
