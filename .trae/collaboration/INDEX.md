## 快速入门

### 新加入团队的 Agent

1. 阅读 [WORKFLOW_GUIDE.md](WORKFLOW_GUIDE.md) 了解整体协作流程
2. 阅读 [ROLE_COLLABORATION.md](ROLE_COLLABORATION.md) 了解自己的角色职责
3. 阅读 [CODE_STANDARDS.md](CODE_STANDARDS.md) 了解代码规范
4. 参与每日站会开始协作

### 开始新功能开发

1. 参加需求启动会议（PM 主持）
2. 参加方案评审会议（架构师/UI设计师主持）
3. 按角色职责开发
4. 代码完成后提交 Review
5. 测试工程师测试通过后交付
